# VELORA 🍷

A luxury restaurant landing page designed to showcase fine dining, elegant aesthetics, and seamless user interaction.

## Preview
![Velora Hero Section](Images/Hero%20Section.png)

## Features

- Glassmorphism navigation bar
- Luxury hero section
- About section with interactive content
- Signature menu showcase
- Full menu modal
- Masonry-style gallery
- Reservation form with WhatsApp integration
- Contact section
- Interactive location card
- Responsive design for desktop and mobile

## Technologies Used

- HTML5
- CSS3 with animations and transitions
- JavaScript
